# Heart Disease Prediction Web App

This project is a **Machine Learning based Heart Disease Prediction system** with a **Flask web interface**.  
Users can enter health details and the model predicts the **10-year risk of heart disease**.

---

## Features
- Predicts heart disease risk using ML
- User-friendly web interface (Bootstrap)
- Flask backend
- Trained using Scikit-Learn
- Model saved using Pickle

---

##  Machine Learning Models Used
- Random Forest Classifier
- Decision Tree Classifier
- StandardScaler for data preprocessing

---

##  Technologies Used
- Python
- Flask
- Scikit-Learn
- Pandas & NumPy
- HTML, CSS
- Bootstrap
